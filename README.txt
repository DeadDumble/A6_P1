README - Assignment 6 Part 1: Undirected Graph Representation and Traversals

Implemented Functions:
----------------------
1. removeEdge(char src, char dest)
   - Removes an undirected edge from both the adjacency list and adjacency matrix.

2. bfsM(char start)
   - Performs Breadth-First Search using the adjacency matrix.

3. dfsL(char start)
   - Performs Depth-First Search using the adjacency list.

4. dfsM(char start)
   - Performs Depth-First Search using the adjacency matrix.
